How to compile, build, and install this application:

1) To host this files you need to have server which supports mysql and php.
2) The folder named "davisdb" should be in the root directory of that server. (Mainly htdocs or www)
3) davisdb.sql script should in run or import into the mysql database.
4) Username, hostname and passoword should be set accordingly in config.php file.
5) The web app can be open after running server by localhost/davisbase/

Technical Dependencies: PHP, Mysql, HTML, CSS, Bootstrap, Javascript.
